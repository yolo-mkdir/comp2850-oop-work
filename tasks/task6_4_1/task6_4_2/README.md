# Task 6.4.2 - Anagram Checker

A Gradle project for unit testing an anagram checking function.

## What are Anagrams?
Anagrams are words or phrases formed by rearranging the letters of another word or phrase.

Examples:
- "listen" ↔ "silent"
- "funeral" ↔ "real fun"
- "school master" ↔ "the classroom"
- "debit card" ↔ "bad credit"

## How to Run

### Run Tests
./gradlew test

### Run Main Program
./gradlew run

### Build Project
./gradlew build
